# Github repository list

A Github repository listing built with React JS

*This project was built as a challenge for the GoStack Bootcamp by rocketseat.com.br

# Use
### Instal dependencies
`yarn`

### Enjoy !
`yarn start`

# Development
### Instal dependencies
`yarn`

### May the code be with you !
`yarn dev`
